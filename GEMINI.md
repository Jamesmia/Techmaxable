# Project Overview

This is a web project named Techmaxable that uses Tailwind CSS for styling. The project has been set up and deployed to Azure Static Web Apps.

# Building and Running

## Building

To build the CSS, run the following command from the `Techmaxable` directory:

```bash
npx tailwindcss -i src/input.css -o dist/output.css
```

## Running

This project is deployed as an Azure Static Web App. You can access it at the provided URL after deployment.

## Testing

This project does not have any tests yet.

# Development Conventions

This project uses Tailwind CSS. All new styles should be implemented using Tailwind classes.
